# Railgun

**The Advanced Visual Discord Bot Builder.**

Railgun is a powerful desktop application that allows you to create fully-featured Discord bots without writing a single line of boilerplate code. Built with modern web technologies, it offers a visual node-based editor to design commands, events, and interactions.

![Railgun Interface](https://github.com/repyh/railgun/blob/main/docs/assets/main.png?raw=true)

## üöÄ Key Features

*   **Visual Node Editor**: Powered by Rete.js, design your bot's logic by connecting nodes visually.
*   **Discord.js v14**: Fully compatible with the latest Discord.js features.
*   **Rich Interactions**: Native support for Slash Commands (`/cmd`), Buttons, Select Menus, and Modals.
*   **Embed Builder**: Visually design beautiful embeds with real-time preview.
*   **Event Handling**: Easy listeners for events like `ClientReady`, `InteractionCreate`, and `MessageCreate`.
*   **One-Click Build**: Export your project as a standalone, runnable Node.js bot application.

## üì¶ Getting Started

### Prerequisites

*   [Bun](https://bun.sh) (latest version)
*   [Node.js](https://nodejs.org/) (required for Electron)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/railgun.git
    cd railgun
    ```

2.  Install dependencies:
    ```bash
    bun install
    ```

3.  Run in development mode:
    ```bash
    bun run dev
    ```
