# Contributing to Railgun

Thank you for your interest in Railgun!

**Please Note:** This project is currently **maintained by a closed team**. We have made the source code available for transparency and educational purposes (under the **CC BY-NC-ND 4.0** License).

## Policy on Contributions
We are **not** actively seeking external contributions or Pull Requests at this time.
- Implementation details and architectural decisions are currently in flux.
- Unsolicited Pull Requests may be closed without review.

## Reporting Issues
If you encounter a critical bug or security vulnerability:
1.  Please check existing issues to avoid duplicates.
2.  Open a new Issue describing the problem, reproduction steps, and expected behavior.

## Forking
You are welcome to fork this repository and modify it for your *private* use, subject to the terms of the **CC BY-NC-ND 4.0** license.

**Redistribution of modified versions is prohibited.**
